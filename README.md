# blog
个人学习笔记


# create a new repository on the command line
```sh
echo "# test" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:yuanduobao/blog.git
git push -u origin main
```

# push an existing repository from the command line
```
git remote add origin git@github.com/yuanduobao/blog.git
git branch -M main
git push -u origin main
```
